# hospital-management-sql
Hospital Management System using PostgreSQL
This project is a Hospital Management Database built using PostgreSQL. It manages patient records, doctor information, appointments, and prescriptions.

## Features
- Database Creation
- Four Tables
- CRUD Operations
- Filtering Queries
- Joins
- Aggregate Functions
- GROUP BY & HAVING
- Views
- Indexes
- Subqueries
- Final Reports

- PostgreSQL
- SQL
